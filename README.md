# Formulário Geolocalizado com Mapa Interativo

Este projeto permite que o usuário selecione um ponto no mapa, insira um comentário e salve a informação no Firebase — tornando a experiência fácil, visual e interativa.

## Descrição

Com interface moderna e responsiva, este formulário foca na experiência do usuário: basta clicar no mapa, preencher o comentário e enviar. As coordenadas são preenchidas automaticamente, e mensagens de feedback aparecem para indicar sucesso ou erro. O projeto pode ser facilmente adaptado para coleta de informações geográficas, sugestões ou pesquisas locais[web:1].

## Instalação

- Clone este repositório:  
  `git clone https://github.com/seuusuario/formulario-mapa.git`
- Instale dependências (se necessário) e configure seu projeto do Firebase.
- Adicione seu arquivo de configuração Firebase em `/api/firebaseConfig`.
- Abra o arquivo HTML em qualquer navegador moderno.

## Uso

- Clique no mapa para selecionar uma localização.
- Escreva seu comentário no campo apropriado.
- As coordenadas selecionadas aparecem automaticamente nos campos da interface.
- Clique em "Enviar Comentário" para registrar a informação.

## Experiência do Usuário

O formulário proporciona uma interação simples e intuitiva, tornando fácil para qualquer pessoa enviar dados geolocalizados. Feedbacks visuais reforçam cada etapa, aumentando o conforto e a confiança do usuário ao participar.
